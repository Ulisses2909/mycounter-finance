# MyCounter — Personal Finance App

Aplicativo completo de controle financeiro pessoal com:
- 📊 Dashboard interativo com gráficos
- 💰 Controle de transações (receitas/despesas/transferências)
- 🎯 Orçamentos por categoria
- 📁 Importação de CSV
- 🤖 Assistente IA (Fin)
- ⚙️ Configurações e exportação de dados
- ⭐ **Finance PRO** — Curso completo de finanças pessoais

## Acesse o app
🔗 **URL do app:** [https://mycounter-finance.vercel.app](https://mycounter-finance.vercel.app)

## Como usar
1. Abra o link acima no navegador
2. Crie uma conta ou faça login
3. Comece a registrar suas transações
4. Explore o Finance PRO para aprender mais!

## Tecnologias
- HTML5 + CSS3 + JavaScript vanilla
- Canvas API para gráficos
- localStorage para persistência
- PWA-ready
